Entramos en https://dev.mysql.com/downloads/connector/j/
Creamos al carpeta lib 
   
    MiProyectoJava/
    ├── src/
    │   └── Main.java
    ├── bin/        # carpeta donde se guardarán los .class
    └── lib/
        └── mysql-connector-j-9.4.0.jar

control + `= abre terminal, escribimos: javac -cp lib/mysql-connector-j-9.4.0.jar -d bin src/Main.java
java -cp bin:lib/mysql-connector-j-9.4.0.jar Main

Creamos carpeta src, (MAIN.java dentro) y bin y lanzamos
javac src/Main.java -d bin
java -cp bin Main

En la linea Class.forName añadimos el cj : Class.forName("com.mysql.cj.jdbc.Driver"); SI ES INFERIOR EL DRIVER A v.5 SI NO BORRAR
